# Venom Tests

- ## Pre-requisite
    - [Install venom](https://github.com/ovh/venom#installing)
    - Have a running instance of tyk and operator.
    - Port forward tyk-gateway svc to port 8080
    

- ## Run tests
    cd venom-tests
    venom run
